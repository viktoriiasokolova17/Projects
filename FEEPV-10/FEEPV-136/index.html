Атрибут allow в елементі iframe використовується для визначення, які функції або API браузер повинен дозволити для вбудованого вмісту. Параметри:
accelerometer: дозволяє вбудованому вмісту доступ до даних акселерометра пристрою, що вимірює прискорення
autoplay: дозволяє автоматичне відтворення медіа без взаємодії користувача.
clipboard-write: дозволяє вбудованому вмісту писати в буфер обміну
encrypted-media: дозволяє вбудованому вмісту відтворювати зашифровані медіа файли
gyroscope: дозволяє вбудованому вмісту доступ до даних гіроскопа пристрою, який вимірює орієнтацію.
picture-in-picture: дозволяє вбудованому відео використовувати режим "картинка в картинці"
web-share: дозволяє вбудованому вмісту використовувати API для спільного доступу в інтернеті
